# vorto Homebrew Tap

Homebrew formulae for [vorto](https://github.com/vorto-editor/vorto), a
Vim-flavored modal terminal editor with batteries included.

## Install

```sh
brew install vorto-editor/tap/vorto
```

Or tap first, then install:

```sh
brew tap vorto-editor/tap
brew install vorto
```

## Upgrade

```sh
brew update
brew upgrade vorto
```

## Supported platforms

Pre-built binaries are provided for:

- macOS (Apple Silicon / arm64)
- Linux x86_64
- Linux arm64

Intel macOS is not currently published as a pre-built binary; install from
source with `cargo install vorto` instead.
